# 保姆级 Notion 个人设计作品网站搭建教程

---

## 前言：为什么用 Notion 做作品集？

| 优势 | 说明 |
|------|------|
| 免费 | 基础版完全免费 |
| 零代码 | 不需要编程基础 |
| 快速上线 | 10分钟搭好框架 |
| 可当 CMS | 作品更新直接在 Notion 编辑 |
| 可绑定域名 | 有域名可以绑定，没域名直接用 Notion 链接 |

---

## 第一步：注册 & 基础设置

### 1.1 注册账号
- 访问 **notion.so** → 用 Google 账号或邮箱注册
- 建议用 **Gmail 或企业邮箱**，方便管理

### 1.2 认识工作台
- 左侧是**侧边栏**，管理所有页面
- 右上角是**分享/发布**按钮
- 空白页面中央的灰色字是**占位提示**，不影响使用

---

## 第二步：搭建作品集主页

### 2.1 新建主页面
1. 左侧点 **"+"** 新建页面
2. 页面命名建议：`作品集 / Portfolio / 你的名字`
3. 不用选模板，直接从空白页面开始

### 2.2 主页必备模块

按这个结构填：

```
[你的头像/名字]

一句话介绍
（比如：专注于支付体验的 UX 设计师）

📁 项目作品
  - 项目1名称 + 封面图
  - 项目2名称 + 封面图
  - 项目3名称 + 封面图

💼 关于我
  个人简介段落...

🔗 联系方式
  Email / LinkedIn / 你的其他链接
```

### 2.3 页面内加图片
- 在文字下方按 **回车** 新行
- 输入 `/image` → 上传本地图片或粘贴链接
- 图片会自动适应页面宽度

---

## 第三步：创建单个项目页面

### 3.1 新建项目子页面
在主页项目列表每项下面：
1. 把文字**拖动**变子页面（或者按 `/page` 创建链接页面）
2. 建议命名：`项目名 - 案例`

### 3.2 项目页面的结构

```
[封面大图 - 拖满宽度]

项目名称
一句话描述

📋 项目背景
  描述项目背景、目标、你解决的问题

🎯 我的角色
  你是 UX、UI 还是全程参与

📐 设计流程
  用户研究 / 需求分析 / 方案设计 / 测试验证

[中间穿插设计图]

🔗 链接
  Figma 原型链接（可选）

[返回上级页面的链接]
```

---

## 第四步：让页面好看一点

### 4.1 基础排版规则
- **标题层级**：用 `/h1` `/h2` `/h3` 不要全用正文
- **段落**：每段不超过 3-4 行，阅读更舒服
- **图片间距**：图片和文字之间要回车留白

### 4.2 常用块指令
| 指令 | 效果 |
|------|------|
| `/h1` | 大标题 |
| `/h2` | 二级标题 |
| `/h3` | 三级标题 |
| `/image` | 添加图片 |
| `/divider` | 分割线 |
| `/quote` | 引用块 |
| `/callout` | 高亮提示框 |
| `/toggle` | 折叠展开块 |

### 4.3 快速格式化
- 选中文字后出现工具栏，可以加**粗体**、*斜体*、链接、颜色
- 不需要记快捷键

---

## 第五步：发布为网站

### 5.1 开启分享链接
1. 右上角点 **分享** → **发布**
2. 选择要发布的页面（主页会包含子页面）
3. 复制生成的 **公开链接**

### 5.2 链接格式
```
notion.so/你的名字/页面id
```

### 5.3 绑定自定义域名（可选）
- 需要 Notion Plus 版（付费）
- 在**设置 → 域名**里配置
- 需要一个域名（如 yourname.com）

---

## 第六步：持续维护

### 6.1 更新频率
- 建议每完成一个项目就加一个
- 旧项目可以**归档**（删除或移走）

### 6.2 保护隐私
- 发布前检查是否有敏感信息
- 联系方式确认是否愿意公开

---

## 加分技巧：让作品集更专业

### 技巧 1：用封面图提升视觉
- 封面图尺寸建议 **1200×630px**（社交分享图比例）
- 用 Figma 或 Canva 快速制作
- 风格统一，整个作品集保持一致的色调

### 技巧 2：加"关于我"页
- 教育背景 / 工作经历
- 技能标签（如：Figma / 用户研究 / 设计系统）
- 个人照片（建议用职业照）

### 技巧 3：项目加上数据成果
- "提升了 30% 转化率"
- "用户满意度从 3.2 升到 4.5"
- 有数据的设计案例更有说服力

---

## 常见问题

**Q：免费版能发布几个页面？**
A：免费版可以发布**无限页面**，但有部分限制（主要是子页面数量）

**Q：别人能看到我未发布的内容吗？**
A：不能，只有点了发布的页面才能被公开访问

**Q：能嵌入 Figma 原型吗？**
A：可以，用 `/embed` 输入 Figma 链接，就能内嵌预览
